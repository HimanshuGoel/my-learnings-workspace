Topic 5 — Transformer Architecture

Summary: The Transformer stacks attention + feed-forward + normalization into a repeating block, forming the backbone of every modern LLM.

Why It Exists

  • Problem — attention alone is linear (weighted average). Need non-linearity, stability, and scalable structure.
  • Solution — repeating block: [Attention → Add&Norm → FFN → Add&Norm]. Stack N times. Simple pattern that scales from 100M to 1T params.
  • Mental Model — layered middleware pipeline. Each layer processes → transforms → passes to next. 32 layers = 32 levels of abstraction.

The Transformer Block

One Transformer block. Stack N of these (N=12 small, N=32 medium, N=96 large).

Components Explained

ComponentWhat It DoesWhy Needed
Multi-Head AttentionEach token gathers context from all othersCore relationship computation
Feed-Forward (FFN)2 linear layers + activation (expands then contracts)Non-linearity + "knowledge storage" (most params live here)
Layer NormNormalizes activations to stable rangePrevents training collapse in deep networks
Residual ConnectionAdds input directly to output (skip)Prevents gradient vanishing, enables deep stacking
Positional EncodingAdds position info to embeddingsAttention is order-agnostic — needs position signal

Encoder vs Decoder

VariantModelsAttention TypeUse Case
Encoder-onlyBERTBidirectional (sees all tokens)Classification, embeddings, NER
Decoder-onlyGPT, LLaMA, ClaudeCausal (sees only past tokens)Text generation (dominant today)
Encoder-DecoderT5, BARTEncoder=bidirectional, Decoder=causalTranslation, summarization
Key insight: Most modern LLMs (GPT-4, Claude, LLaMA) are decoder-only. They generate text left-to-right, one token at a time, using causal (masked) attention.

Common Misconceptions

WrongCorrect
"The Transformer is one thing"Encoder, decoder, or both — GPT ≠ BERT architecturally
"More layers = smarter"Diminishing returns. More layers = more memory + slower
"FFN is unimportant"FFN has MOST of the parameters — stores learned "knowledge"
"Position doesn't matter"Without positional encoding, "I love you" = "you love I"
