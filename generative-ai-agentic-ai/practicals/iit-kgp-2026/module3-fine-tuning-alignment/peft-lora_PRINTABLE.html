Module 3, Topic 3 — PEFT & LoRA

LoRA (Low-Rank Adaptation) freezes ALL base model weights and trains only tiny adapter matrices that capture task-specific adjustments. It achieves 90-98% of full fine-tuning quality while training just 0.1% of parameters — making fine-tuning accessible on consumer GPUs. This is your DEFAULT fine-tuning method. Full SFT is the exception; LoRA is the rule.

Problem: Full SFT requires massive GPUs and creates full model copies per task. LoRA makes fine-tuning practical: smaller GPUs, faster training, composable adapters, nearly the same quality.

How LoRA Works

LoRA adds a small "bypass" path alongside each frozen weight matrix. Only the bypass (A×B) is trained.

Implementation — LoRA with PEFT

from peft import LoraConfig, get_peft_model, TaskType lora_config = LoraConfig( task_type=TaskType.CAUSAL_LM, r=16, # rank (start here) lora_alpha=32, # scaling (usually 2× rank) lora_dropout=0.05, # regularization target_modules=["q_proj","v_proj","k_proj","o_proj", # attention "gate_proj","up_proj","down_proj"], # MLP ) model = get_peft_model(model, lora_config) model.print_trainable_parameters() # → trainable: 6.5M || total: 8B || trainable%: 0.08%

from transformers import BitsAndBytesConfig bnb_config = BitsAndBytesConfig( load_in_4bit=True, bnb_4bit_quant_type="nf4", bnb_4bit_compute_dtype=torch.bfloat16, bnb_4bit_use_double_quant=True, ) model = AutoModelForCausalLM.from_pretrained(model_name, quantization_config=bnb_config) model = get_peft_model(model, lora_config) # Same LoRA config as above # 8B model now fits on 16GB GPU!

Training (TRL — Recommended)

from trl import SFTTrainer, SFTConfig config = SFTConfig( output_dir="./lora-output", num_train_epochs=3, per_device_train_batch_size=4, gradient_accumulation_steps=4, learning_rate=2e-4, # 10× higher than full SFT! max_seq_length=2048, bf16=True, ) trainer = SFTTrainer(model=model_name, peft_config=lora_config, train_dataset=..., args=config) trainer.train() trainer.save_model("./lora-adapter") # Just the adapter (few MB)

Loading & Swapping Adapters

# Load base + adapter: model = PeftModel.from_pretrained(base_model, "./lora-adapter") # Merge for simpler deployment: merged = model.merge_and_unload() # Multiple adapters on same base: model.load_adapter("./adapter-task-B", adapter_name="task_b") model.set_adapter("default") # Task A model.set_adapter("task_b") # Task B — same base, different behavior!
