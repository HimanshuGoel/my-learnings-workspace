Module 4, Topic 3 — Agent Architectures

Agent architectures define how an LLM system reasons, acts, and loops. The simplest is ReAct (reason-act-observe loop). More complex patterns add planning, reflection, or multiple agents. The architecture choice is the foundational decision — start simple with ReAct (handles 80% of tasks), add complexity only when measured quality is insufficient for your specific task class.

Problem Statement: A raw LLM can't take multi-step actions. Agent architectures provide the loop structure that lets an LLM reason about what to do, execute an action (tool call), observe the result, and decide whether to continue or stop. Choosing the wrong architecture wastes cost on simple tasks or fails on complex ones.

Decision Flowchart

  • Is the task simple (1-3 tool calls)? → YES → ReAct or Tool-Calling Agent
  • Does it need a multi-step plan? → YES → Plan-and-Execute
  • Can subtasks run in parallel? → YES → LLM Compiler
  • Does quality matter more than speed? → YES → Reflexion (self-critique loop)
  • None of the above? → Plan-and-Execute with step budget

Implementation: ReAct Agent (LangGraph)

from langgraph.graph import StateGraph, END from langgraph.prebuilt import ToolNode from langchain_openai import ChatOpenAI from langchain_core.tools import tool from typing import TypedDict, Annotated from langgraph.graph.message import add_messages class AgentState(TypedDict): messages: Annotated[list, add_messages] @tool def search_database(query: str) -> str: """Search the internal knowledge base for information.""" return f"Found 3 results for: {query}" @tool def calculate(expression: str) -> str: """Evaluate a mathematical expression.""" return str(eval(expression)) tools = [search_database, calculate] llm = ChatOpenAI(model="gpt-4o", temperature=0).bind_tools(tools) def call_model(state: AgentState) -> dict: return {"messages": [llm.invoke(state["messages"])]} def should_continue(state: AgentState) -> str: last = state["messages"][-1] if last.tool_calls: return "tools" return END workflow = StateGraph(AgentState) workflow.add_node("agent", call_model) workflow.add_node("tools", ToolNode(tools)) workflow.set_entry_point("agent") workflow.add_conditional_edges("agent", should_continue, {"tools": "tools", END: END}) workflow.add_edge("tools", "agent") app = workflow.compile()

Step Budget (Safety Guard)

MAX_ITERATIONS = 10 def should_continue(state: AgentState) -> str: tool_call_count = sum(1 for m in state["messages"] if hasattr(m, "tool_calls") and m.tool_calls) if tool_call_count >= MAX_ITERATIONS: return END # Force stop — prevent infinite loops if state["messages"][-1].tool_calls: return "tools" return END
