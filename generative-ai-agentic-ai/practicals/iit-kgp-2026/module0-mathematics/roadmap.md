## Phased Learning Sequence

### Phase 1 — Intuition

Sets → Functions → Vectors → Dot Product → Matrix basics → Probability → Statistics

### Phase 2 — AI Mental Model

Embeddings → Semantic space → Cosine similarity → Matrix multiplication → Conditional probability → MSE

### Phase 3 — Deep Learning Math

Gradients → Derivatives → Chain rule → Optimization → CNN intuition → Transformers intuition

### Phase 4 — Engineering

NumPy → Pandas → PyTorch → GPU/CUDA → Docker
